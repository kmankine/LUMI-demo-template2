# LUMI AI Factory branded template

This is a template to create a [Jekyll] site that:

- has LUMI AI Factory branding and colours;
- can be built and published on [GitHub Pages];
- can be built and previewed locally, and published on other platforms;
- is built on top of [Just the Docs] theme.

To get started with creating a site, simply:

1. Click "[use this template]" to create a GitHub repository, name it appropriately for the materials
2. Go to Settings > Pages > Build and deployment > Source, and select GitHub Actions
3. If you see a green tick appear next to your name in the repository, the website has been deployed correctly. If it’s a yellow circle, it’s still building.
4. Once you see the green tick, click the Gear Icon next to "About" on the right side. Check the box "Use your GitHub Pages website" and click Save. This puts the live link right at the top of the repo so you never have to remember the URL!
5. Edit the `index.md` file and add your content there. The website will be rebuilt with your new content automatically.
6. To force refresh the website page (to tell your browser not to use the cached version), press ctrl + F5 on Linux and Windows, or Cmd + Shift + R on Mac.

[^1]: [It can take up to 10 minutes for changes to your site to publish after you push the changes to GitHub](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/creating-a-github-pages-site-with-jekyll#creating-your-site).

Every time you push a commit to github, the website will be redeployed.
